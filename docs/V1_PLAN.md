# Version 1 Build Plan

## Goal

Launch a clean MVP for university students that can accept study material and turn it into understanding, revision help, and exam practice.

## Pages

1. Landing page
2. Login page
3. Signup page
4. Dashboard
5. Study workspace
6. Test page
7. Result page

## Phase 1

- scaffold Next.js app
- install Tailwind CSS
- set up app router structure
- create shared layout
- initialize git workflow

## Phase 2

- design landing page
- design login page
- design signup page
- create dashboard shell

## Phase 3

- set up Firebase auth
- implement login
- implement signup
- protect dashboard routes

## Phase 4

- add PDF upload input
- add topic input
- add course outline input
- add plain text input
- store user study sessions

## Phase 5

- connect OpenAI
- build summarize action
- build quick revision action
- build explanation modes
- build question generation

## Phase 6

- implement MCQ test flow
- implement short-answer flow
- implement theory/writing flow
- add score view
- add model answers

## Phase 7

- add feedback system
- add recent history
- polish responsive UI
- prepare for initial deploy

## Notes

- keep V1 focused
- avoid analytics and payments for now
- prioritize strong UX and clarity
